A tag HTML CODE serve para indicar que o conteúdo é um pequeno fragmento de código, estilizando-o de forma apropriada. Por padrão, o conteúdo é exibido usando a fonte monoespaçada padrão do user agent's. As tags HTML são instruções que informam ao navegador como formatar o texto e a estrutura de uma página web. Elas podem ser usadas para criar títulos, parágrafos, imagens, links, entre outros. Para usar tags HTML corretamente, é preciso seguir algumas regras: Uma tag HTML deve ter uma tag de abertura, conteúdo e uma tag de fechamento. A tag de abertura começa com um símbolo < > e a tag de fechamento termina com um símbolo . Algumas tags HTML podem ser abertas, o que significa que não precisam ser fechadas com .
